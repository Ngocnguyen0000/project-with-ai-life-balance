# Life Balance — Android (Kotlin + Jetpack Compose)

Source files only — no Gradle wrapper/`build.gradle` yet (needs to be generated by Android Studio itself).

## Setup
1. Android Studio → **New Project → Empty Activity (Compose)**. Package name: `com.lifebalance.app`, minimum SDK 26+.
2. Copy the `app/src/main/java/com/lifebalance/app/` folder from this repo over the generated one.
3. Add the **Nunito Sans** font files (`.ttf`, from [Google Fonts](https://fonts.google.com/specimen/Nunito+Sans)) into `res/font/` as `nunito_sans_regular.ttf`, `nunito_sans_medium.ttf`, `nunito_sans_semibold.ttf`, `nunito_sans_bold.ttf` — `designsystem/Type.kt` references these exact names via `R.font.*`.
4. Add dependencies in `app/build.gradle.kts`: `androidx.compose.material3`, `androidx.compose.material:material-icons-extended`, `androidx.activity:activity-compose`.
5. Add the **Supabase Kotlin SDK** (`io.github.jan-tennert.supabase`) before wiring real auth/data (see step 9 / backend consultation).

## Structure
- `designsystem/` — Color.kt, Type.kt, Theme.kt (mirrors `design-system/life-balance/MASTER.md` — keep in sync)
- `components/` — Buttons.kt, TextFields.kt, Cards.kt (reusable, used across all screens)
- `screens/auth/` — LoginScreen, SignUpScreen, ForgotPasswordScreen
- `screens/lifebalance/` — RadarChart.kt (Canvas-drawn) + LifeBalanceOverviewScreen.kt (home screen, Epic 7)
- `MainActivity.kt` — app entry, auth gate, bottom nav (5 tabs matching Figma)
